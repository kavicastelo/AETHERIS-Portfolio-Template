# 🌌 AETHERIS: Futuristic HUD-Style Developer Portfolio

Aetheris is an advanced, premium, and cinematic personal portfolio engine engineered for **Senior Developers, Software Architects, and AI Engineers**. Designed with the visual language of high-performance operating systems (HUDs), tactical control panels, and cyberpunk environments, Aetheris turns your professional history and system projects into an immersive, interactive digital ecosystem.

Built with bleeding-edge technology (**Next.js 16**, **React 19**, **Tailwind CSS v4**, and **Framer Motion 12**), it features an **interactive client-side Data Analysis Hub** and built-in **Answer Engine Optimization (AEO)** mechanisms to maximize your search authority in AI search queries.

## 🚀 Quick Start Guide

Get your environment up and running in seconds.

### 1. Prerequisites
Ensure you have [Node.js (v18.x or newer)](https://nodejs.org) installed on your system.

### 2. Installation
Clone the folder or extract the zip archive and install the dependencies:
```bash
npm install
```

### 3. Spin up Local Development
Launch the interactive preview server with hot reloading:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) inside your web browser.

### 4. Build & Export Production Files
Generate the static production build:
```bash
npm run build
```
This generates the optimized static files in the [`/out`](file:///e:/PROJECTS/react/portfolio-2026/out) directory, ready for drag-and-drop hosting anywhere.

## 🎨 Theme Customization Summary

Accents and glows can be altered in a single place. Open [`src/styles/globals.css`](file:///e:/PROJECTS/react/portfolio-2026/src/styles/globals.css) and customize the color hex variables:

```css
@theme {
  --color-neon-cyan: #00f3ff;    /* Change to primary brand color (e.g. orange #ff6b00) */
  --color-neon-magenta: #ff00ff; /* Change to secondary accent color */
}
```
